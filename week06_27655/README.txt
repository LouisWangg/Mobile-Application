Apabila user menekan tombol A, lalu menekan layar handphonenya maka hasil yang dikeluarkan
adalah sebuah lingkaran berwarna hijau yang semakin membesar dan menutupi sebagian besar
layar user.

Sedangkan apabila user menekan tombol B dan menekan gambar kuda yang terdapat di tengah
layar handphonenya, maka kuda yang sebelumnya tidak bergerak akan menjadi bergerak
dengan kecepatan yang telah ditentukan program.

Lalu dengan menekan tombol C, user dapat menekan 2 button / icon yang terdapat di layar, yaitu
Spring dan Fling. Apabila user menekan Spring, maka button tersebut akan melompat ke bawah
dan kembali ke tempat semula. Semakin sering user menekan tombol Spring, maka lama kelamaan
button tersebut akan berpindah posisi semakin menurun. Sedangkan apabila user menakan tombol
Fling, maka button tersebut akan berputar dari atas ke bawah dan akan berhenti dalam beberapa
saat. Untuk memutarkan lagi button tersebut, user dapat mengklik lagi button Fling.
