Berdasarkan skenario-skenario yang telah dicoba, setiap kali terjadi pengiriman data dan 
button telah ditekan, maka data yang sebelumnya telah dikirimkan akan menghilang dan 
digantikan dengan data yang baru. Sehingga melalui aplikasi ini, data yang sebelumnya telah
diketik dan dikirimkan tidak akan tersimpan ataupun dipaparkan di layar user. Lalu dengan
adanya fitur kirim dan balas, maka melalui aplikasi tersebut user satu dapat mengirimkan
pesan kepada user dua, dan user dua dapat mengirimkan balasan pesan kepada user satu.
