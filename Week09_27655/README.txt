Dengan aplikasi tersebut, pengguna dapat membuat sebuah database yang berisikan nim, nama, 
email, dan nomor hp mahasiswa. Setelah berhasil menambahkan data, data tersebut akan ditampilkan
di halaman utama aplikasi.
Apabila pengguna menambahkan data dengan menggunakan NIM yang sama dengan data lainnya, maka
data tersebut akan menjadi invalid dan tidak akan ditampilkan pada halaman utama aplikasi.
Apabila pengguna melakukan swipe ke kiri pada data yang terdapat di halaman utama, maka data
tersebut akan terhapus.
Sedangkan apabila swipe dilakukan ke arah sebelah kanan, maka aplikasi akan berganti halaman
dan pengguna dapat melakukan edit pada data tersebut. Nilai-nilai yang dapat diubah adalah 
nama, email, dan nomor hp saja.
Apabila pengguna menekan "Hapus Semua" yang terdapat pada action bar dengan gambar titik tiga,
maka seluruh data pada halaman utama akan dihapus.
