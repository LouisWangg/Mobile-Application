Aplikasi ini akan menampilkan sebuah map dengan menggunakan google map API yang diambil
pada url file XML google_maps_api.
Terdapat 4 jenis titik mark yang dapat digunakan pada map, yaitu marker, circle, polyline, 
dan polygon.
Terdapat juga button "TAMBAH" untuk membuat tanda lokasi baru.
Ketika menggeser map, maka akan muncul tempat baru sesuai dengan tampilan map pada device.
- Marker untuk menandakan titik lokasi dan bisa memeberi nama pada titik tersebut.
- Circle untuk membuat lingkaran yang melingkari titik tertentu. (Dapat mengatur radius 
  untuk lingkaran tersebut)
- Polyline untuk memberi garis dari satu titik ke titik yang lain. (Bisa banyak titik)
- Polygon untuk membentuk suatu bentuk yang menandai suatu tempat.
